# 修复 UETasksQueryExtensions.cs 编译错误

## 任务概述
**目标**: 修复 `UETasksQueryExtensions.cs` 中的两个编译错误（CS0411 和 CS1061）

**动机**: 解决编译失败问题，使 C# 项目能够正常构建

**影响范围**: 仅修改 `UETasksQueryExtensions.cs` 文件

## 问题分析

### 错误 1: CS0411 - 无法推断类型参数
```
Error CS0411: 无法从用法中推断出方法"UETasksQueryExtensions.CollectChunks<T>(IQuery)"的类型参数。
```
**原因**: `CollectChunks<T>` 返回 `(T[] chunks, int count)`，但返回的元组中 `T[]` 没有被直接使用，导致无法推断泛型类型 `T`。

### 错误 2: CS1061 - IQuery 没有 ArchetypeChunk
```
Error CS1061: "IQuery"未包含"ArchetypeChunk"的定义
```
**原因**: `IQuery<T>` 有 `ArchetypeChunk` 属性，但方法使用 `IQuery`（非泛型版本）。

## 修复方案

将泛型方法 `CollectChunks<T>` 拆分为两个具体类型的重载方法：

```csharp
// T1 版本
private static (ArchetypeChunk<T1>[] chunks, int count) CollectChunks<T1>(this IQuery<T1> query)
    where T1 : struct
{
    var chunkCount = query.ChunkCount;
    var list = new List<ArchetypeChunk<T1>>(chunkCount > 0 ? chunkCount : 0);

    foreach (var chunk in query.ArchetypeChunk)
    {
        list.Add(chunk);
    }

    var chunks = list.ToArray();
    return (chunks, chunks.Length);
}

// T1, T2 版本
private static (ArchetypeChunk<T1, T2>[] chunks, int count) CollectChunks<T1, T2>(this IQuery<T1, T2> query)
    where T1 : struct
    where T2 : struct
{
    var chunkCount = query.ChunkCount;
    var list = new List<ArchetypeChunk<T1, T2>>(chunkCount > 0 ? chunkCount : 0);

    foreach (var chunk in query.ArchetypeChunk)
    {
        list.Add(chunk);
    }

    var chunks = list.ToArray();
    return (chunks, chunks.Length);
}
```
