# 任务概述
- 目标：移除 UETasks JobHandle 的 Combine 相关逻辑，避免额外性能开销，并清理 MainMenu 的 Combine 性能用例。
- 影响范围：UE Tasks 句柄层与测试入口。
- 边界：不改动 Schedule/Execute 任务调度核心路径。

# 聊天与讨论背景
- 用户明确要求不保留任何 Combine 逻辑，倾向开发者自行管理 handle 列表。
- 目标是减少额外合并开销并简化调用链。

# 设计思路
- 删除 Combine 的 C# 工具类与句柄支持方法。
- 删除 C++ CombineHandles 内部调用与绑定。
- 清理 MainMenu 中 Combine 测试用例与输出。
